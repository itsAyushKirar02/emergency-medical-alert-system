# Emergency Medical Alert System

A Java-based web application designed to streamline emergency medical responses. The system allows users to send instant alerts, provides real-time mapping for hospitals, and automates notifications through SMS and Email.

##  Features
* **Google OAuth 2.0:** Secure user authentication and sign-in.
* **Real-time Maps:** Integration with Google Maps API to locate nearby medical facilities.
* **Automated Alerts:** SMS notifications via Twilio and email alerts via Gmail API.
* **Secure Architecture:** Sensitive credentials (API keys, DB passwords) are managed via Environment Variables to prevent security leaks.

##  Tech Stack
* **Backend:** Java (Servlets & JSP)
* **Database:** MySQL (Local & Cloud SQL compatible)
* **APIs:** Google Maps, Gmail API, Twilio SMS API
* **DevOps:** Git for version control, Environment Variables for secret management
